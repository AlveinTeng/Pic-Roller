Global Options Available
Property Default Type Description
pictures [] Array The array to store all the pictures you want to show
Picture_clickable false Boolean The boolean to decide whether your pictures can be clicked
buttons [] Array The arrray to store your buttons
autoPlay false Boolean The boolean to decide whether your picture can be autoplayed
actived 0 Int Store the index of current actived picture
clicked 0 Int Store the index of the picrure/button being clicked
next 0 Int Indicating the index of the next picture/button to be actived
moveHorizontal false Boolean The boolean to decide whether the pictures are moving vertically or horizontally
speed 3000 Float The value that decides how fast the pictures transimited
multipic 1 Int The number of pictures that being transimitted at one time
hoverOnPause false Boolean The boolean to decide whether an autoplayed scroller can be paused once the mose is on image
Methods in picRoller 
      
  \\ Add inner text of button
  \\ button: int, text: string
  picRoller.addButtonText(button, text)  	

  \\ Set the speed of the picture transmitting
  \\ speed: Float 
  picRoller.setSpeed(speed)		
  
  \\ Set the number of pictures being transmitted at one time
  \\ mutipic: int
  picRoller.setMutiPic(mutipic)		
  
  \\ Decide whether the picture move horizontally or vertically
  \\ horizontal: boolean
  picRoller.horizontal(horizontal)		
  
  \\ Decide whether the picture can be auto played
  \\ autoplay: boolean
  picRoller.auto(autoplay)			
  
  \\ Decide whether the pictures can be clicked to move the pictures
  \\ clickable: boolean
  picRoller.clickable(clickable)
  
  \\ To enable the hover on pause effect on autoplayed pictures
  picRoller.hoverPause()

  \\ To display those pictures in the picture roller
  \\ displayRegion: DOM element, buttonType: string
  picRoller.displayPictures(displayRegion, buttonType)
